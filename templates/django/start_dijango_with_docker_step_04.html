Các bước tạo file .gitignore trên Windows

Để tạo một file .gitignore trên Windows, bạn có thể làm theo các bước sau:

Bước 1: Tạo file .gitignore trong Command Prompt

  1. Mở Command Prompt hoặc PowerShell.
  2. Điều hướng đến thư mục dự án của bạn:
    3. cd C:\Users\ADMIN\Desktop\ICONS\GITHUB\DOCKER_ALL_PROJECTS\django-docker-TAG\myproject
  3. Chạy lệnh để tạo file .gitignore trống:
    4. echo.>.gitignore

Bước 2: Mở và chỉnh sửa file .gitignore

Tiếp theo, mở file .gitignore với Notepad hoặc trình soạn thảo văn bản bất kỳ:

notepad .gitignore

Sau đó, dán nội dung sau vào file .gitignore:

# Các file Python bytecode
__pycache__/
*.pyc
*.pyo
*.pyd
*.pyc

# Thư mục môi trường ảo (virtual environment)
venv/
env/
.venv/
.virtualenv/
pip-log.txt
pip-delete-this-directory.txt

# Thư mục Django
db.sqlite3
*.log
*.pot
*.pyc
*.mo
*.db
*.sqlite3

# Thư mục media
/media/

# Thư mục static
/static/

# Các file cấu hình của hệ thống
.idea/
.vscode/

# Các file của OS
.DS_Store
Thumbs.db

# Thư mục của các công cụ quản lý dự án (IDE, editor)
*.swp
*.swo

# Các file cài đặt của editor như Sublime Text, Atom, etc.
*.sublime-workspace
*.sublime-project

Bước 3: Lưu file và commit vào Git

  1. Sau khi chỉnh sửa xong, lưu file .gitignore.
  2. Thêm file vào Git và commit thay đổi:
    3. git add .gitignore
    git commit -m "Add .gitignore file"

Vậy là bạn đã tạo và thêm file .gitignore vào dự án Git của mình.
