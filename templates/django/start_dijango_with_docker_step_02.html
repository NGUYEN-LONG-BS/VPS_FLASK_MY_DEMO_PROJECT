Hướng Dẫn Build và Chạy Docker Container từ Dockerfile

Bước 1: Kiểm tra Dockerfile

Đảm bảo rằng file Dockerfile của bạn đã được lưu đúng tên và ở đúng thư mục mà bạn muốn sử dụng. Tên của file phải là Dockerfile mà không có đuôi .txt hay bất kỳ đuôi nào khác.

Bước 2: Mở Command Prompt hoặc Terminal

  1. Mở Command Prompt (Windows) hoặc Terminal (macOS, Linux).
  2. Điều hướng đến thư mục chứa file Dockerfile bằng lệnh cd. Ví dụ, nếu file Dockerfile của bạn nằm trong thư mục C:\Users\ADMIN\Desktop\ICONS\GITHUB\DOCKER_ALL_PROJECTS\django-docker-TAG, bạn gõ:
    3. cd C:\Users\ADMIN\Desktop\ICONS\GITHUB\DOCKER_ALL_PROJECTS\django-docker-TAG

    Lưu ý: Hãy chắc chắn rằng bạn đang đứng trong thư mục chứa Dockerfile.

Bước 3: Xây dựng Docker Image từ Dockerfile

Sử dụng lệnh sau để build Docker image từ Dockerfile:

docker build -t my-django-app .

Giải thích:

  • -t my-django-app: Đây là tên bạn muốn đặt cho Docker image. Bạn có thể thay đổi "my-django-app" thành tên tùy ý của mình.
  • .: Dấu chấm cuối cùng chỉ định rằng Dockerfile nằm trong thư mục hiện tại.

Lệnh trên sẽ đọc Dockerfile trong thư mục hiện tại và xây dựng một Docker image từ đó. Quá trình này có thể mất một chút thời gian tùy thuộc vào các bước trong Dockerfile và tốc độ mạng (đặc biệt là khi tải các image từ Docker Hub).

Bước 4: Kiểm tra Docker Image đã được tạo thành công

Sau khi build xong, bạn có thể kiểm tra xem image của mình đã được tạo ra hay chưa bằng lệnh:

docker images

Lệnh này sẽ liệt kê tất cả các Docker images trên hệ thống của bạn. Tìm tên của image bạn vừa tạo trong danh sách.

Bước 5: Chạy Docker Container

Khi Docker image đã được tạo thành công, bạn có thể chạy container từ image đó bằng lệnh sau:

docker run -p 8000:8000 my-django-app

Giải thích:

  • -p 8000:8000: Điều này sẽ map cổng 8000 trên container tới cổng 8000 trên máy tính của bạn. Bạn có thể truy cập ứng dụng Django của mình qua http://localhost:8000 trong trình duyệt.
  • my-django-app: Đây là tên của Docker image bạn đã tạo.

Bước 6: Truy cập ứng dụng Django

Mở trình duyệt web và nhập vào URL sau:

http://localhost:8000

Bạn sẽ thấy trang chào mừng mặc định của Django nếu mọi thứ đã được cấu hình chính xác.

Bước 7: Quản lý Container

  • Để dừng container: Bạn có thể sử dụng lệnh: 
    docker stop <container_id>
    Bạn có thể tìm ID của container bằng cách sử dụng lệnh docker ps để liệt kê các container đang chạy.
  • Để xóa container: Sau khi dừng container, bạn có thể xóa container bằng lệnh: 
    docker rm <container_id>

Tóm tắt

  • Tạo và chỉnh sửa Dockerfile xong, bạn chỉ cần build Docker image bằng lệnh docker build.
  • Sau đó, chạy container từ image bằng lệnh docker run.
  • Kiểm tra ứng dụng Django trong trình duyệt qua http://localhost:8000.

Sau khi làm các bước trên, ứng dụng Django của bạn sẽ chạy trong container Docker. Nếu có thêm vấn đề nào, đừng ngần ngại hỏi mình!
